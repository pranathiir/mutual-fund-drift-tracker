# Mutual Fund Drift Tracker

A Python tool for tracking and analyzing portfolio drift in mutual funds, helping investors monitor how their fund allocations deviate from target asset allocations over time.

## Overview

The Mutual Fund Drift Tracker helps you:
- Monitor how your mutual fund portfolio's actual allocation drifts from your target allocation
- Identify when rebalancing is needed
- Analyze historical drift patterns
- Make data-driven investment decisions

## Features

- **Portfolio Monitoring**: Track current vs. target allocations
- **Drift Analysis**: Quantify how far your portfolio has drifted
- **Rebalancing Insights**: Get recommendations on when and how to rebalance
- **Historical Tracking**: View drift trends over time

## Requirements

- Python 3.7+
- See `requirements.txt` for dependencies
